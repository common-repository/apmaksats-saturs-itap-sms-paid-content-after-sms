== Description ==
 Shows content after paid with sms. This plugin are connected to ITAP hosting (https://itap.lv). You should make profile in your personal profile section.
 
 Parāda satura pēc tam, kad tas apmaksāts. Nepieciešams izveidot savu profilu, lai iegūtu API un klienta id, savā Itap kontā.

== Installation ==

1. Extract the archive file into your plugins directory in the itapaidcontent folder.
2. Activate the plugin in the Plugin options.


1. Atarhivēt spraudni jūs spraudņu mapē zem itapaidcontent mapes.
2. Aktvizēt spraudni.
